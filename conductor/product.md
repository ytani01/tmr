# Product Definition: tmr

## Purpose
ターミナル上で動作する、効率的でカスタマイズ可能なポモドーロ・タイマーを提供し、開発者やCLIユーザーの集中力維持を支援する。

## Target Audience
- ターミナルで作業を完結させたい開発者
- シンプルで軽量な集中ツールを求めるCLIユーザー
- ポモドーロ・テクニックを厳格に管理・実践したいユーザー

## Core Values
- **Efficiency**: ターミナルから離れずに、コマンド一つで時間管理を開始できる。
- **Flexibility**: 作業時間や休憩時間をユーザーのニーズに合わせて柔軟に設定可能。
- **Focus**: TUI（Text User Interface）を活用した視覚的なフィードバックにより、作業への没入感を高める。

## Key Features
- **Timer Management**: タイマーの開始、停止、一時停止、再開。
- **Customizable Sessions**: 25分/5分などの標準プリセットに加え、任意の時間を設定可能。
- **Visual Feedback**: 動的なプログレスバーによる進捗表示（TUIスタイル）。
- **Notifications**: セッション終了時の通知（ターミナル出力、デスクトップ通知、音など）。
