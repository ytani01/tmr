# Initial Concept
ターミナル上で動作する、効率的でカスタマイズ可能なタイマーです。

# Product Guide - tmr

## Vision
`tmr` は、ターミナル環境で作業を完結させるパワーユーザーのために設計された、軽量かつレスポンシブな CLI タイマーおよびポモドーロ・タイマーです。シンプルでありながら柔軟な操作性と、視覚的に優れた TUI フィードバックを両立し、作業の集中力維持を強力にサポートします。

## Target Audience
- ターミナルのサイズや環境にこだわりのあるパワーユーザー
- シンプルかつレスポンシブな CLI ツールを好む層
- 作業コンテキストを切り替えずに時間管理を行いたい開発者や学生

## Core Goals
- **優れた視覚表示**: ターミナルのリサイズにリアルタイムで追従し、常に最適な情報量を表示するレスポンシブなプログレスバーを提供します。
- **柔軟な操作性**: タイマー稼働中の一時停止、時間の微調整（進める・戻る）を直感的なキー操作で実現します。
- **軽量かつ迅速**: 最小限の依存関係で動作し、コマンド一つで即座にタイマーを開始できる機動力を持たせます。

## Key Features
- **Simple Timer**: 分単位で指定可能なカウントダウンタイマー。
- **Pomodoro Timer**: 作業、短い休憩、長い休憩のサイクルを管理するポモドーロ機能。
- **Responsive UI**: 画面幅に合わせて表示項目を動的に選択し、レイアウト崩れを防ぐ堅牢な TUI。
- **Interactive Control**: 一時停止、リセット、時間のスキップ/バックをサポート。

## Design Philosophy
- **Simplicity**: 設定に時間を取られず、インストール直後から直感的に利用できること。
- **Flexibility**: ユーザーの状況に合わせて動作中にタイマーを微調整できる自由度を提供すること。
- **Robustness**: 多様なターミナル環境において、安定した表示と動作を維持すること。
